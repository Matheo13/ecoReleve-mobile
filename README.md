## Requirements

Cordova: 6.0.0
Node.js: 6.5.0 (Supports ES6)

## Usage

Clone the repository.

### Browser development
`cordova run browser -- --live-reload`

### Mobile
```
cordova run android -- --lr
cordova run ios -- --lr
```
